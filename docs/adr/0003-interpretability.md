# ADR 0003: Prefer Interpretability Over Black-Box Complexity

## Status

Accepted

## Context

The system is intended for decision support and daily brief generation. Users need to understand why a regime or factor is favored.

## Decision

Prefer interpretable features, transparent scores, documented assumptions, and simple validation before more complex models.

## Consequences

The platform may sacrifice some raw predictive power. It gains auditability, explainability, and easier model governance.
